# 🚀 E-Commerce Website

This project is a modern Full-Stack-Style ecommerce frontend built with the latest web technologies. The goal is to demonstrate how to create a clean, responsive 
and production-ready shopping experience using Next.js 15, Tailwind CSS v4, Stripe and Zustand without relying on a traditional backend database such as Prisma, PostgreSQL or Neon.

The platform focuses on frontend architecture, state management and secure payment integration. 
Product data and checkout flows are handled directly through Stripe making the application lightweight while still supporting real-world ecommerce functionality.

# ⚙️ Tech Stack
- **Next.js 15** –
Utilizes server components modern routing and optimized rendering for high performance.

- **Tailwind CSS v4** –
Implements a CSS first configuration for fast responsive and consistent UI styling.

- **TypeScript** –
Provides strong type safety better tooling and improved maintainability.

- **Stripe** –
Handles product data checkout sessions and secure payment processing.

- **Zustand** –
Manages global client side state efficiently with minimal boilerplate.

# ⚡ Features
- **🛍️ Dynamic Product Carousel** –
A visually engaging landing page carousel that automatically cycles through featured products.

- **📱 Responsive Product Pages** –
Fully responsive product detail pages with interactive quantity controls for a smooth shopping experience.

- **🛒 Real-Time Cart Updates** –
Live cart state synchronized across components using Zustand including an always-up-to-date navbar cart icon.

- **💳 Secure Stripe Checkout** –
A seamless and secure checkout flow powered entirely by Stripe’s API.

- **🎨 Modern UI Design** –
A clean professional interface built with Tailwind CSS v4 and shadcn-inspired components.
